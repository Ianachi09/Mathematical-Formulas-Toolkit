A Program that lists you formulas which allows you to see the formula with a few clicks and use them. The program is also dynamic so you can add new one within the program. THis is the first project that I thought about doing as I thought this idea solves a real-world problems such as not being able to know what formula is that without searching for it on online.
I also tried implementing ofstream which makes the program more dynamic and parser so that I can add new formulas without changing the hard code of the program.
